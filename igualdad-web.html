Perfiles y Funciones

  • 01 Ajuste de voz y navegación

    Los usuarios ciegos y con baja visión, así como los que tienen problemas de movilidad, confían en sus teclados -y no en un ratón- para navegar por los sitios web. Al entrar en un sitio web, pulsarán la tecla TAB y recibirán la notificación de que la herramienta de accesibilidad está instalada para facilitar la navegación. Estas funciones ayudan a las personas con discapacidad a navegar por las páginas del sitio web de forma correcta y semántica.

  • 02 Ajustes de color

    Las funciones de ajuste del color ayudan a las personas con deficiencias visuales como el daltonismo y otras discapacidades visuales. Estas importantísimas funciones de accesibilidad permiten al usuario ver el sitio web en cualquier variación de color de su elección. Igualdad Web permite a los usuarios con deficiencias visuales ajustar el diseño y la interfaz de usuario del sitio web, de modo que satisfaga perfectamente sus necesidades de accesibilidad.

  • 03 Ajustes de contenido

    Las funciones permiten a los usuarios con distintas discapacidades mejorar el contenido del sitio web de forma automática y ajustar el contenido de su sitio web para que satisfaga perfectamente las necesidades de accesibilidad según las directrices WCAG 2.2 en el nivel AA+. Desde aumentar el tamaño de las fuentes, cambiar los estilos de letra, ajustar el interlineado, resaltar titulares y mucho más.
